image2sheet.danieltesting.space contains 3 different tools
1. A quick way to add multiple images to a page for printing
2. Default settings for playing card printing
3. An upscaler for printing A3-A0 using A4 or A3 paper

<ol>
<li>Upload multiple images and let it auto fit them to the page</li>
<li>You can change page size, orintation, how many images should be on each page and the gap between them</li>
<li>You can drag the images around, duplicate or delete</li>
<li>Add a backing image for double sided prints</li>
<li>Then export it to PDF</li>
</ol>
