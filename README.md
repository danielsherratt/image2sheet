https://image2sheet.danieltesting.space contains 3 different tools
1. A quick way to add multiple images to a page for printing
2. Default settings for printing playing card 
3. An upscaler for printing A3-A0 using A4 or A3 paper


**How?**
1. Upload multiple images and let it auto fit them to the page
2. You can change page size, orintation, how many images should be on each page and the gap between them
3. You can drag the images around, duplicate or delete
4. Add a backing image for double sided prints
5. Then export it to PDF and print
